## Project Overview

The **Employee Database Management System** serves as a reliable structure for storing and retrieving detailed employee data, such as job titles, salaries, departments, and management structures. This schema is designed for HR tasks and reporting needs, making it straightforward to query and analyze employee-related information.

## Database Schema

The database schema consists of the following tables:

* **Employees** : Contains each employee’s unique ID, name, title, gender, birth date, and hire date.
* **Titles** : Lists job titles, which connect to employees through a foreign key.
* **Salaries** : Tracks salary information for employees.
* **Departments** : Holds details about each department in the organization.
* **Department Employees** : Links employees to their respective departments.
* **Department Managers** : Maps managers to departments, establishing department heads.
